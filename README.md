# CodeAlpha Pakistan Air Quality & Health Risk Dashboard

## Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship.

The project analyzes updated air quality data for Pakistani cities from March to June 2026. It includes city-wise AQI comparison, PM2.5 pollution trends, hourly pollution patterns, health risk classification, web scraping of air pollution news headlines, and sentiment analysis.

## Tasks Completed

* Task 1: Web Scraping
* Task 2: Exploratory Data Analysis
* Task 3: Data Visualization
* Task 4: Sentiment Analysis

## Tools Used

* Python
* Google Colab
* Pandas
* Plotly
* BeautifulSoup
* NLTK VADER Sentiment Analyzer
* Open-Meteo Air Quality API

## Key Features

* Collected updated air quality data using a public API
* Analyzed AQI and PM2.5 pollution levels across Pakistani cities
* Created city-wise and monthly pollution visualizations
* Classified AQI into health risk categories
* Scraped Pakistan air pollution and smog-related news headlines
* Applied sentiment analysis on scraped headlines

## Files Included

* `Updated_Air_Quality_Data_March_to_June_2026.ipynb`
* `updated_pakistan_air_quality_mar_jun_2026.csv`
* `updated_city_aqi_summary_mar_jun_2026.csv`
* `scraped_air_pollution_news_sentiment.csv`

## Conclusion

This project shows how data analytics can be used to understand environmental pollution patterns, identify high-risk cities, and analyze news sentiment around smog and air pollution.
